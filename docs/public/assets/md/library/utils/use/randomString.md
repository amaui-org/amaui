
# randomString

Returns a random string of provided length, using lowercase, uppercase letters, special symbols & numbers.

```ts
randomString(7);

// yad/4au
```

## API

#### randomString

```ts
// defaults
// length: 1e1
const randomString: (length_?: number) => string;
```
