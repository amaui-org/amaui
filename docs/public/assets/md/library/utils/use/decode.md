
# decode

Decodes any string value from base64 value into its actual value type.

Value will be deserialized into its value type.

```ts
decode('eyJhIjoxMTE0fQ==');

// { a: 1114 }
```


## API

#### decode

```ts
const decode: (value_: string) => any;
```
