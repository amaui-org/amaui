
# updateQueryParams

Updates query param keys with provided values.

```ts
updateQueryParams({ a: 114 });

window.location.search;

// ?a=114
```

## API

#### updateQueryParams

```ts
const updateQueryParams: (value?: {}, override?: boolean) => void;
```
