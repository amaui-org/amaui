
# shuffle

Randomly shuffles the string value, provided number of times.

Each shuffle, 2 random charaters in the string swap places.

```ts
shuffle('yvryq6zG', 14);

// yyG6qvzr
```

## API

#### shuffle

```ts
// defaults
// toShuffle: 14
const shuffle: <T extends unknown>(value: string | T[], toShuffle?: number) => string | T[];
```
