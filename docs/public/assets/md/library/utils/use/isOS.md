
# isOS

Used to validate an operating system.

```ts
isOS('mobile');

// true
```

### Type

- mac
- mobile
- android
- ios
- windows
- linux

### value

User provided value for the method.

## API

#### isOS

```ts
default function isOS(type: string, value?: any): any;
```
