
# pick

Picks characters from a string, in length of between min & max values.

```ts
pick('abcdefghijklmnopqrstuvwxyz', 3, 4);

// auyi
```

## API

#### pick

```ts
const pick: (value: string, min?: number, max?: number) => string;
```
