
# getTimezoneOffset

Returns timezone offset.

```ts
const amauiDate = new AmauiDate();

getTimezoneOffset(amauiDate);

// +02:00
```

## API

#### getTimezoneOffset

```ts
default function getTimezoneOffset(value?: AmauiDate, divider?: string): string;
```
