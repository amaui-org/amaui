
# bubbleSort

Bubble sort method.

### Options

#### ascending

In what direction to sort, default true.

### Example

```ts
const value = [1, 1114, 114, 14];

bubbleSort(value);
```

## API

#### bubbleSort

```ts
default function bubbleSort<T = number>(value: Array<T>, ascending?: boolean): Array<T>;
```
