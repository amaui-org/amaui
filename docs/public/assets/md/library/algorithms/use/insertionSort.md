
# insertionSort

Insertion sort method.

### Options

#### ascending

In what direction to sort, default true.

### Example

```ts
const value = [1, 1114, 114, 14];

insertionSort(value);
```


## API

#### insertionSort

```ts
default function insertionSort<T = number>(value: Array<T>, ascending?: boolean): Array<T>;
```
