
# wait

Using await waits for provided number of milliseconds.

```ts
await wait(1114);

// Waited for 1114 milliseconds
```

## API

#### wait

```ts
// defaults
// milliseconds: 400
const wait: (milliseconds?: number) => Promise<undefined>;
```
