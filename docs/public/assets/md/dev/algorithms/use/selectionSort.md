
# selectionSort

Selection sort method.

### Options

#### ascending

In what direction to sort, default true.

### Example

```ts
const value = [1, 1114, 114, 14];

selectionSort(value);
```


## API

#### selectionSort

```ts
default function selectionSort<T = number>(value: Array<T>, ascending?: boolean): Array<T>;
```
