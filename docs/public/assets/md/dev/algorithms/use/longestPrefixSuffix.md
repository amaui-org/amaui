
# longestPrefixSuffix

Finds longest repeating prefix, and how far repeat reached on each character.

### Example

```ts
longestPrefixSuffix('asdasuasday');

// [0, 0, 0, 1, 2, 0, 1, 2, 3, 4, 0]
```

## API

#### longestPrefixSuffix

```ts
default function longestPrefixSuffix(value: string): Array<number>;
```
