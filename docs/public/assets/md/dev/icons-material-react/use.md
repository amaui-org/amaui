### Version

There are 18k+ icons including various usefull versions:
- Rounded
- Rounded Filled
- Rounded W100
- Rounded W100 Filled
- Sharp
- Sharp Filled
- Sharp W100
- Sharp W100 Filled
- Two Tone

### Width

- All icons without `W100` are regular width of 400.
- W100 is width 100.

### Other

All icons are using `@amaui/ui-react`'s `Icon` UI element.
