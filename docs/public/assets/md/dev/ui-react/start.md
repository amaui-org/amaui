
### Add

```bash
// yarn
yarn add @amaui/ui-react

// npm
npm install @amaui/ui-react
```

### Use

```jsx
// Import any of the ui
import { Button } from '@amaui/ui-react';

function A(props) {

  return (
    <Button>{props.children}</Button>
  );
}
```

### Dev

Install

```bash
yarn
```

Test

```bash
yarn test
```

### Prod

Build

```bash
yarn build
```
