## Getting started

### Add

```sh
  // yarn
  yarn add @amaui/ui-react

  // npm
  npm install @amaui/ui-react
```

### Use

```javascript
  // Import any of the ui
  import { Button } from '@amaui/ui-react';

  function A(props) {

    return (
      <Button>
        {props.children}
      </Button>
    );
  }
```

### Dev

Install

```sh
  yarn
```

Test

```sh
  yarn test
```

### Prod

Build

```sh
  yarn build
```
